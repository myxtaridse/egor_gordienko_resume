# Egor Gordienko — Motion Design Portfolio

[**🔗 Ссылка на сайт**](https://myxtaridse.github.io/egor_gordienko_resume/)

Минималистичный одностраничный сайт-портфолио моушн-дизайнера.

## Техническая реализация

* **HTML5 & CSS3**: Чистая семантическая верстка без сторонних библиотек.
* **Layout**:
    * **CSS Grid**: Двухколоночная сетка для основных секций контента. 
    * **Flexbox**: Навигация в хедере и адаптивная раскладка контактов в футере. 
* **Video**: Адаптивный плеер Vimeo с соотношением сторон 16:9.
* **Adaptive**: Корректное отображение на разных экранах (от мобильных до десктопов).
